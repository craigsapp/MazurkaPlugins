Click on the red dynamic library entries to download a compiled plugin for a specific operating system. Click on the green plugin name to view the documentation for the plugin.

Warning: Delete any vamp 0.9 plugins from the "Vamp Plugins" directory before running Sonic Visualiser 1.0, or SV will not startup. Historic vamp 0.9 compliant plugins can be found on this page.

Linux

    Save any of the following .so files into the directory /usr/local/lib/vamp if you are a superuser. If not, then save to your home directory in ~/vamp. If you want to download them from the terminal, type the following command: 

             wget http://sv.mazurka.org.uk/download/linux/yyy
    where yyy is one of the .so dynamic library files listed in red below.
